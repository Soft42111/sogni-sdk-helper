# Sogni SDK AI Helper

A premium, high-fidelity AI Studio built with the Sogni SDK. This application provides a decentralized AI platform for chat, image synthesis, and multimodal tasks, featuring a clean modular architecture and robust local persistence.

## Features
- **Sogni Supernet Integration**: Powered by decentralized GPU workers.
- **Modular Architecture**: Clean separation of logic into components (Sidebar, ChatArea, MessageBubble).
- **Advanced Markdown**: Full support for code syntax highlighting and interactive widgets.
- **Persistence**: Local storage synchronization for sessions and authentication.
- **Account Management**: Support for API Keys, User Login, and Direct Registration.

## Tech Stack
- React 19
- Vite
- Framer Motion (Animations)
- Lucide React (Icons)
- Sogni SDK (@sogni-ai/sogni-client)

## Getting Started
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Run dev server: `npm run dev`.
